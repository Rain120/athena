# Resources

A few resources that are helpful for building with Slate.

# Libraries

These libraries are helpful when developing with Slate:

  • is-hotkey (opens new window) is a simple way to check whether an onKeyDown handler should fire for a given hotkey, handling cross-platform concerns like cmd vs. ctrl keys for you automatically.

# Extensions and Plugins

These extensions and plugins add additional features and capabilities to Slate:

  • Plate (opens new window) Rich text editor plugin system for Slate & React
  • slate-angular (opens new window) Angular-based view layer, which is a useful supplement to Slate for building a rich text editor using Angular.
  • slate-yjs (opens new window) Collaborative editing utilities for Slate leveraging Yjs
  • slate-collaborative (opens new window) Collaborative editing utilities for Slate leveraging Automerge

# Editors

These pre-packaged editors are built on top of Slate, and can be helpful to see how you might structure your code:

  • Accord Project Markdown Editor (opens new window) is a WYSIWYG editor for CommonMark (opens new window).
  • Canner Editor (opens new window) is a rich text editor.
  • Chatterslate (opens new window) helps teach language grammar and more at Chatterbug (opens new window).
  • CoCalc (opens new window) Collaborative Calculation editor in the Cloud
  • French Press Editor (opens new window) is a customizeable editor with offline support.
  • Nossas Editor (opens new window) is a drop-in WYSIWYG editor.
  • React Force Slate Editor (opens new window) is a light-weight medium-style editor with no editor chrome.
  • React Page (opens new window) is a self-contained, customizable inline WYSIWYG editor library.
  • Slate Plugins Next (opens new window) provides an editor with configurable and extendable plugins.

(Or, if you have their exact use case, can be a drop-in editor for you.)
